Angular Password Input

The cFormPassword is a fully featured Angular password input field built with Bootstrap and CoreUI. It includes built-in password visibility toggling, sm / lg sizing options, and support for disabled, readOnly, floating label and form validation states. Ideal for login forms, signup screens, and any secure input in your Angular app.

Angular Password Input Sizing

The cFormPassword supports sizing property to adjust the input field size. Use sizing="lg" for large inputs, sizing="sm" for small inputs, or leave it out for default sizing.
